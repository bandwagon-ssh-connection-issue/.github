## 误区一："连不上SSH = IP被墙，必须换IP"

这是传播最广的误解，也是最贵的误解。

**真相：** IP被墙只是众多原因之一，而且它是最后才需要考虑的情况，不是第一反应。

SSH连不上的常见原因，按发生概率排序大概是这样的：

1. **VPS没开机或宕机** — 频率最高，最容易被忽视
2. **端口号填错** — 搬瓦工的SSH端口是随机5位数，不是22
3. **密码输错或密码忘了** — 账号用root（全小写），密码在购买邮件里
4. **本地网络/防火墙问题** — 换个网络环境试试
5. **TCP端口被封** — 能ping通但SSH连不上
6. **IP被完全封锁** — 国内国外都ping不通

大部分人卡在第1-3条，但上来就怀疑第6条。

**正确姿势：先登录KiwiVM面板确认VPS状态，再一步步排查，别急着掏钱换IP。**

## 误区二："VPS能ping通，SSH就应该能连上"

这个逻辑听起来很合理，但它在技术上是错的。

**真相：** ping用的是ICMP协议，SSH用的是TCP协议。这是两条完全不同的通路。

国内某些情况下，ICMP包被放行，TCP包被丢弃，结果就是：ping通，SSH连不上。

这种现象有个名字叫**TCP阻断**，是端口层面的封锁，不是IP被完全封。

遇到这种情况怎么办？

**第一步：** 去KiwiVM面板，用官方自带的IP检测工具（或者访问 tcp.ping.pe），把你的IP和SSH端口输进去测一下：
- 国内TCP红，国外TCP绿 → 端口被封或IP被墙（需进一步区分）
- ICMP通，TCP全红 → TCP阻断，考虑换SSH端口或迁机房
- 国内国外都红 → IP被完全封，或VPS宕机

**第二步：** 如果是TCP阻断，先尝试在KiwiVM的 Root shell – interactive 里把SSH端口改掉（改成一个不常用的端口）。很多时候改完端口就能连上了。

**第三步：** 如果换端口还不行，在KiwiVM面板里免费迁移一个机房，换了IP就换了网络路径，通常能解决。

只有在以上都不管用的情况下，才考虑付费换IP。

## 误区三："SSH端口就是22"

这个误区专坑新手。

**真相：** 大多数VPS的默认SSH端口确实是22，但**搬瓦工不一样**。

搬瓦工的SSH端口是一个随机生成的5位数，购买后会发到你的注册邮箱里。如果你不小心用22端口去连，当然连不上。

怎么找到正确的SSH端口？

- 去注册邮箱里找购买确认邮件，里面有IP、端口和初始密码
- 或者登录 bwh81.net → My Services → 找到你的VPS → 点进KiwiVM Control Panel → 首页就能看到SSH端口号

**另外要注意的：** 用户名必须是小写的 `root`，不是 `Root`，不是 `admin`，就是 `root`。很多人在这里栽跟头。

## 误区四："VPS显示Active就说明在运行"

**真相：** 搬瓦工的状态分两层——官网后台显示的Active是账户状态，KiwiVM面板里显示的Running才是VPS真正的运行状态。

有时候VPS本身宕机了，但官网后台显示的还是Active。

正确的检查步骤：

1. 登录 bwh81.net → Services → My Services，确认VPS显示 **Active**（账户正常，没被暂停）
2. 点进 KiwiVM Control Panel，查看 Status 一栏，必须是 **Running** 才算真正开机
3. 如果是 Stopped，点 Start 开机，等30秒左右再试SSH

还有一种情况：VPS因内存爆满或负载过高死机了。这时候直接在KiwiVM面板里点 Stop（不是Restart），等状态变成Stopped之后再点Start重新开机，比直接重启更彻底。

## 误区五："忘了密码就没办法了，只能重装系统"

**真相：** 搬瓦工有专门的一键重置密码功能，完全不需要重装系统。

步骤很简单：

1. 确保VPS是**开机状态**（Running），这一步很多人反了，以为要先关机
2. 登录KiwiVM面板
3. 左侧菜单找到 **Root password modification**
4. 点击 **Generate and set new root password**
5. 新密码就出来了，复制好用于SSH登录

注意：旧密码立即失效，新密码是随机生成的字符串，记得保存下来。如果想改成自己记得住的密码，可以连上SSH之后再用 `passwd` 命令修改。

如果重置密码提示错误"Failed to reset root password (739102)"，检查一下VPS是否真的处于Running状态，有时候显示Running但实际还没完全启动，等个半分钟再试。
